Hello,

Cave Boy is a simple toy, maybe even a game.

Move cave boy, find secret, go exit.

Use arrow keys or joystick.

To play play Cave Boy, you will need to have LOVE <http://love2d.org> installed. Then cd to this directory and execute `love .`

Read author info in authors.txt

Read license info in license.txt

The homepage of the game is at <http://wiki.github.com/qubodup/cave-boy/cave-boy>
