Hello,

Cave Boy is a simple toy, maybe even a game.

Move Cave Boy to find secret. Then, go exit.

Use arrow keys or joystick.

To play play Cave Boy, you will need to have LOVE <love2d.org> installed. Then cd to this directory and execute `love .`

Read author info in authors.txt

Read license info in license.txt
